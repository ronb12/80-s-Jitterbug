# Delete duplicate 80s Jitterbug Firebase projects

**Keep:** **jitterbug80s** (80s Jitterbug) — this is the one the app uses.

**Delete these two duplicates:**

1. **fir-jitterbug** (80'S Jitterbug)  
   → [Open project → Project settings → Delete project](https://console.firebase.google.com/project/fir-jitterbug/settings/general)

2. **jitterbug-80s** (80's Jitterbug)  
   → [Open project → Project settings → Delete project](https://console.firebase.google.com/project/jitterbug-80s/settings/general)

**Steps for each:** Open the link → scroll to the bottom → click **Delete project** → check the boxes → confirm. Deletion is permanent (restore possible within 30 days via [Google Cloud Console](https://console.cloud.google.com) if needed).

The Firebase CLI cannot delete projects; this must be done in the Firebase Console.
